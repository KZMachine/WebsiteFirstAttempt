This Website!
 This website was an individual project for my CSCE 315 class. I built this website from scratch using a mix of HTML5, CSS, and a little bit of JavaScript. Prior to this project, I had no experience with web development, however, I was able to pick up on the syntax fairly quickly. For some reason, I felt the need to give it a cloud-based theme. Maybe it was the fact that I had read an article on cloud computing before beginning this project, or maybe it was because I simply love the sky and the color blue. Nevertheless, I wanted the website to feel fun and satisfying to navigate through.
